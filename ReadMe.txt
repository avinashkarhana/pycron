V 1.0

Features :
Checks for file availability in a location and if file do not exist, then restores file's backup from backup directory.
 

Usage :
Just set cron job in scheduler or some other application to run the script periodically as per need.(I can do that if you ask.)

Requirements :

Just a python compiler with libraries named path lib,shutil,datetime (can be easily installed if not.)
